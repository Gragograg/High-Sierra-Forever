  • Change some default settings.
  • Remove stuff that don't work anymore.

    Delete Apps You Don't need

    If you try Taking a System app like iBooks and Putting it into Trash it will say "The item “iBooks” can’t be moved to the Trash because it can’t be deleted." Finder is wrong becuase you can but you need to use Terminal.

    I don't play Chess. My MacBook doesn't have a DVD Drive, and I don't own any Books on iBooks. So, on a fresh install of High Sierra, I like to open the Terminal and run:

    sudo rm -rf "/Applications/Chess.app"
sudo rm -rf "/Applications/DVD Player.app"
sudo rm -rf "/Applications/iBooks.app"

    As You can see everything is managable and you can delete everything with a command just like Linux and it will not break anything. You can Recover apps with Time Machine Later. Just don't Delete something like System Prefrences or anything in Other Folder and you should be good to go

Google Accounts

Go to myaccount.google.com/security and select "2-Step Verification". If 2-Step Verification is turned off, turn it on; there is no way to avoid this. Then, select the "app passwords" option and follow the instructions to create an app password.

Next, open System Preferences → Internet Accounts and sign into Google using your app password in lieu of your normal password.

Certain unusual Google accounts require an alternate sign in procedure. If you are using an app password but still can't seem to sign in, follow the below instructions. This is usually not necessary.

1. Email 1. In System Preferences → Internet Accounts, click Add Other Account... 2. Select Add a Mail account 3. Enter your full email address and app password where indicated 4. When asked which services to enable, deselect everything except Mail 2. Calendar 1. In System Preferences → Internet Accounts, click Add Other Account... 2. Select Add a CalDAV account 3. Enter your full email address and app password where indicated 4. Set the server address to apidata.googleusercontent.com 5. Enter your full email address and app password where indicated
