High Sierra Forever

macOS 10.13 — still the last great macOS.
Here's how I am going to keep it running beautifully.

Why High Sierra, specifically?

macOS High Sierra, released in 2017, represents a turning point. It's the last version of macOS that shipped with the old HFS+ alongside the new APFS, the last to support 32-bit apps natively, and the last to carry the visual spirit of the original Mac OS X Aqua interface before Apple's relentless flattening campaign fully took hold.

It's fast. It's stable. The menu bar is still dark and dense with information. Finder doesn't try to be iOS. And it runs on a wide range of real hardware — including machines that Apple abandoned years ago but which have decades of life still in them.

This site is a living guide for running macOS 10.13 High Sierra in the present day — keeping it online, keeping it safe, and keeping it yours.

Hardware

Supported Macs

High Sierra officially supports Macs from roughly 2009–2010 onwards, depending on the model. This includes MacBook Pros from late 2009, MacBook Airs from 2010, iMacs from 2009, and Mac Pros from 2010. A full list is available on Apple's support page.

Unsupported Macs (OpenCore Patcher)

Thanks to the OpenCore Legacy Patcher, many older Macs that Apple dropped can still run High Sierra — and beyond. If your Mac is too old for the official installer, OCLP is the first thing to try.

Hackintosh

High Sierra was the last release with reliable, community-tested Hackintosh support for a very wide range of hardware. If you're building or repurposing a PC, it's an excellent target. Dortania's OpenCore Install Guide covers the process in detail.

Obtaining High Sierra

Apple still hosts the High Sierra installer in the Mac App Store. Open the App Store on any Mac and search for "macOS High Sierra", or use this direct link from Apple: 

https://apps.apple.com/us/app/macos-high-sierra/id1246284741

If you're on a newer Mac that can't run High Sierra, you can still download the installer to create a bootable USB. The downloaded app will appear in /Applications.

Creating a Bootable Installer

With a 8GB+ USB drive inserted, run the following in Terminal (replace MyVolume with the name of your USB drive): 

sudo /Applications/Install\ macOS\ High\ Sierra.app/Contents/Resources/createinstallmedia \
  --volume /Volumes/MyVolume

Initial Setup

After a fresh install, the first thing to do is apply all available Apple updates. Open App Store → Updates and install everything. Then reboot and repeat until no updates remain. The final version of High Sierra is 10.13.6 with the supplemental security update from 2020.

System Integrity Protection

SIP is enabled by default and is generally a good thing to keep on. However, some patches and tools on this page require it to be disabled temporarily. To do so, boot into Recovery (hold ⌘R at startup) and run: 

csrutil disable

Re-enable it when you're done patching: csrutil enable

Security Patches

Apple's last official security update for High Sierra was in late 2020. Running it today carries real risk — especially with an outdated browser. The most important mitigations:

  • Never use Safari for general browsing. It is not safe on this OS.
  • Keep your router firmware up to date.
  • Regularly back up to an external drive.
  • Avoid opening untrusted files, especially fonts and PDFs.

Community-maintained patches for specific CVEs will be listed here as they become available.

Web Browsing

The most critical thing to get right on an old OS is your browser. Safari on High Sierra is dangerously outdated and should not be used for any web browsing.

Firefox ESR Recommended

Mozilla maintains Extended Support Releases of Firefox with longer security update cycles. Check which ESR version still supports macOS 10.13 at the time you're reading this — it changes over time. Download directly from Mozilla.

Media Playback

High Sierra's QuickTime Player handles common formats well, but struggles with modern codecs like VP9, AV1, and some HEVC profiles. The best solution is IINA.

IINA Recommended

IINA is a modern, native macOS media player built on top of mpv. It handles virtually every video format and feels right at home on the Mac. Older releases support High Sierra.

iina.io →
VLC

The perennial fallback. VLC plays everything and has macOS releases going back years. The latest VLC 3.x series supports High Sierra.

videolan.org →
